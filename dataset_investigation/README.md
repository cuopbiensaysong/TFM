# Dataset Investigation Workspace

This folder contains scripts and artifacts for:

- Investigating source CSV datasets against the project data dictionary.
- Inspecting the template `data/toy_data.pkl` schema.
- Preprocessing CSV files into `.pkl` dictionaries compatible with the training framework.

Structure:

- `scripts/`: executable Python scripts
- `reports/`: generated JSON/Markdown investigation outputs
- `outputs/`: generated framework-ready `.pkl` files
- `tests/`: tests for preprocessing logic
